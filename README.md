# # Analyse de données Business en Python

## Description du projet

Ce projet regroupe plusieurs exercices Python réalisés autour de la **manipulation de données**, de la **transformation de structures**, des **agrégations** et des **analyses business**.

L'objectif est de pratiquer les structures de données Python :

- Listes (`list`)
- Tuples (`tuple`)
- Dictionnaires (`dict`)
- Dictionnaires imbriqués (`nested dictionaries`)
- Ensembles (`set`)
- Compréhensions de listes
- Fonctions (`def`)
- Menus interactifs (`while`, `input`)

Les exercices simulent des données réelles de ventes, de vendeurs, de produits, de météo et de stocks.
