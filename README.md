#  Python - Data Analysis & Gestion de données

##  Description

Ce dépôt regroupe plusieurs projets Python réalisés autour de la **manipulation des données**, de la **programmation procédurale** et de l'**analyse d'informations**.

Les exercices utilisent différentes structures Python :
- Listes
- Tuples
- Dictionnaires
- Dictionnaires imbriqués
- Sets
- Fonctions
- Menus interactifs

L'objectif est de simuler des cas pratiques proches d'applications réelles : gestion d'étudiants, ventes, stocks et analyses business.


---

#  Gestion des étudiants

## Objectif

Créer une application permettant de gérer les informations des étudiants et leurs résultats.

## Fonctionnalités

- Ajouter un étudiant
- Rechercher un étudiant par :
  - ID
  - Nom
- Ajouter ou supprimer une matière
- Calculer la moyenne d'un étudiant
- Afficher les étudiants selon un seuil de moyenne
- Afficher toutes les informations des étudiants
- Calculer la moyenne par matière

## Notions utilisées

- Listes de dictionnaires
- Fonctions Python
- Recherche dans des structures de données
- Manipulation de tuples
- Utilisation de NumPy pour les calculs de moyenne

#  Analyse des ventes et gestion de stock

## Objectif

Analyser des données commerciales contenant des produits, vendeurs et stocks.

## Fonctionnalités

- Transformation des données brutes en dictionnaires
- Calcul du chiffre d'affaires :
  - Par produit
  - Par vendeur
  - Par catégorie

- Analyse des ventes :
  - Filtrage des ventes importantes
  - Produits vendus par jour
  - Classement des vendeurs

- Gestion du stock :
  - Calcul du stock restant
  - Détection des produits en alerte

#  Analyse Business avec données imbriquées

## Objectif

Manipuler des structures complexes représentant des ventes par date et par ville.

Structure utilisée :


Date
Ville
Ventes
Météo
Vendeurs actifs


## Analyses réalisées

### Ventes

- Extraction des ventes
- Calcul du chiffre d'affaires par ville
- Performance quotidienne
- Ranking des produits


### Analyse météo

- Impact de la pluie sur les ventes
- Température moyenne par ville
- Comparaison température / chiffre d'affaires


### Analyse vendeurs

- Ratio :


CA / nombre de vendeurs


- Intersection des vendeurs entre villes
- Migration des vendeurs


### Utilisation des sets

- Produits communs entre villes
- Vendeurs présents dans plusieurs villes
- Différences entre ensembles
